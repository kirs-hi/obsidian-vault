# Mac 快捷键速查手册

> 修饰键符号对照：⌘ Command、⌥ Option (Alt)、⇧ Shift、⌃ Control、Fn 功能键
> Shift 就是键盘上标有 ⇧ 的键，左右各一个，位于 Caps Lock 上方 / Z 键旁边。

- **⌘** = 主命令键，几乎所有快捷键的基础
- **⌥ (Option)** = 替代/附加功能，给已有快捷键加"变体"
- **⇧ (Shift)** = 扩展选择、反向操作（撤销↔重做）、加强版
- **⌃ (Control)** = 系统控制、终端操作
- **Fn** = 功能键切换，模拟全尺寸键盘的 Home/End/PgUp/PgDn

**规律总结：**
1. 单字母快捷键基本都取英文首字母：C=Copy, S=Save, Q=Quit, F=Find...
2. 加 ⇧ = 反向或加强版：⌘Z撤销 → ⌘⇧Z重做，⌘N新建 → ⌘⇧N新建文件夹
3. 加 ⌥ = 变体/替代：⌘V粘贴 → ⌘⌥V移动，⌘H隐藏 → ⌘⌥H隐藏其他
4. 加 ⌃ = 系统级控制：⌃⌘Q锁屏，⌃⌘F全屏

## 一、每天都在用（最高频）

| 快捷键       | 功能           | 记忆                  |
| --------- | ------------ | ------------------- |
| ⌘ + C     | 复制           | Copy                |
| ⌘ + V     | 粘贴           | V 在 C 旁边，复制后紧接粘贴    |
| ⌘ + ⌥ + V | 剪切文件夹        |                     |
| ⌘ + X     | 剪切文本         | X 像剪刀 ✂️            |
| ⌘ + Z     | 撤销           | Z = 最后一个字母 = 退回最后一步 |
| ⌘ + ⌥ + C | 复制文件路径       | Copy                |
| ⌘ + ⌥ + P | 查看文件路径       | Path                |
| ⌘ + A     | 全选           | All                 |
| ⌘ + S     | 保存           | Save                |
| ⌘ + Q     | 退出应用         | Quit                |
| ⌘ + H     | 隐藏当前应用       | Hide                |
| ⌘ + 空格    | Spotlight 搜索 | 搜索文件/APP            |
| ⌘ + F     | 查找           | Find                |

## 二、文本编辑

### 格式

| 快捷键 | 功能 | 记忆 |
|--------|------|------|
| ⌘ + B | 加粗 | Bold |
| ⌘ + I | 斜体 | Italic |
| ⌘ + U | 下划线 | Underline |
| ⌘ + K | 插入链接 | linK（链接🔗） |

### 光标移动

| 快捷键 | 功能 | 记忆 |
|--------|------|------|
| ⌘ + ← / → | 跳到行首/行尾 | ⌘=最大级别，箭头方向跳到头 |
| ⌘ + ↑ / ↓ | 跳到文档开头/结尾 | 同上，纵向到顶/底 |
| ⌥ + ← / → | 按单词移动 | Option=小级别跳跃 |
| Fn + ← / → | Home / End | Fn 模拟全键盘 |
| Fn + ↑ / ↓ | Page Up / Page Down | 同上 |

### 文本选择

以上移动键 + ⇧ = 选中对应范围（Shift = 扩展选区）：

| 快捷键 | 功能 | 记忆 |
|--------|------|------|
| ⌘ + ⇧ + ← / → | 选中到行首/行尾 | 移动键+Shift=边走边选 |
| ⌘ + ⇧ + ↑ / ↓ | 选中到文档开头/结尾 | 同上 |
| ⌥ + ⇧ + ← / → | 按单词选择 | 同上 |

### 删除

| 快捷键 | 功能 | 记忆 |
|--------|------|------|
| ⌥ + Delete | 删除前一个单词 | Option=按词操作 |
| ⌘ + Delete | 删除到行首 | ⌘=最大范围删除 |
| Fn + Delete | 向后删除（Forward Delete） | Fn 补充缺失的 Del 键 |
| ⌃ + K | 删除到行尾 | Kill（来自 Unix/Emacs） |

## 五、截图与录屏

| 快捷键 | 功能 | 记忆 |
|--------|------|------|
| ⌘ + ⇧ + 3 | 全屏截图 | 3=全(全屏) |
| ⌘ + ⇧ + 4 | 区域截图（拖选） | 4=四边形框选 |
| ⌘ + ⇧ + 4 → 空格 | 窗口截图（点击窗口） | 空格=切换到窗口模式 |
| ⌘ + ⇧ + 5 | 截图/录屏工具栏 | 5=五项全能工具箱 |
| ⌘ + ⇧ + 4 → 按住 ⌥ | 窗口截图去阴影 | Option=去掉额外效果 |

截图默认保存到桌面，可在 ⌘+⇧+5 工具栏的"选项"中修改保存位置。
