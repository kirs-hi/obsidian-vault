## 详细介绍

### 1. Excalidraw ⭐ 强烈推荐

**插件 ID：** `obsidian-excalidraw-plugin`

**是什么：** 在 Obsidian 里直接画手绘风格的图，支持流程图、架构图、思维导图，图可以嵌入到任意笔记里。

**为什么适合你：** 你有大量数仓架构图、血缘图需要画，目前 vault 里已经有很多 PNG 附件，用 Excalidraw 可以直接在 Obsidian 里画并保存为可编辑的源文件，不用再截图。考研复习时画知识框架图也很方便。

**效果预览：**
- 画数仓分层架构图 → 嵌入到项目文档里
- 画 Flink 数据流图 → 放进技术笔记
- 考研知识点思维导图 → 放进 35_考研/

**上手方式：** 安装后新建 `.excalidraw` 文件即可，有中文界面。

---

### 2. Tasks ⭐ 强烈推荐

**插件 ID：** `obsidian-tasks-plugin`

**是什么：** 增强 Obsidian 的任务系统，支持截止日期、优先级、重复任务，并能跨所有笔记汇总查询任务。

**为什么适合你：** 你的日记模板里有 `- [ ] 明天想做的事`，但这些任务分散在每篇日记里，没法统一看。Tasks 可以让你在一个地方看到所有未完成的任务，还能按截止日期排序。

**核心用法：**
```tasks
not done
due before tomorrow
```
放在任意笔记里，就能看到所有即将到期的任务。

**特别适合：** 考研复习计划管理、工作任务跟踪、创业内容发布计划。

---

### 3. QuickAdd ⭐ 推荐

**插件 ID：** `quickadd`

**是什么：** 通过快捷键快速执行预设动作，比如"按 Alt+D 新建今天的日记"、"按 Alt+N 新建一条闪念笔记"。

**为什么适合你：** 你已经有 Templater，但 QuickAdd 更适合做"一键触发"的工作流。比如：
- 快捷键 → 弹出输入框 → 输入标题 → 自动在 `00_收件箱/` 创建笔记并套用模板
- 快捷键 → 弹出输入框 → 输入想法 → 自动追加到今天的日记末尾

**和 Templater 的区别：** Templater 负责模板内容，QuickAdd 负责触发时机和交互流程，两者配合使用效果最好。

---

### 4. Linter ⭐ 推荐

**插件 ID：** `obsidian-linter`

**是什么：** 保存文件时自动格式化笔记，包括：自动补全 frontmatter 字段、统一标题格式、清理多余空行、规范列表缩进等。

**为什么适合你：** 你的日记 frontmatter 格式有时不统一（比如 04-02 的 tags 格式不一样），Linter 可以在每次保存时自动修正，保持整个 vault 格式一致，Dataview 查询也更稳定。

**推荐配置：**
- 保存时自动运行
- 自动添加 `date` 字段（如果没有）
- 统一 tags 格式为 `[tag1, tag2]`

---

### 5. Iconize

**插件 ID：** `obsidian-icon-folder`

**是什么：** 给文件夹和文件添加自定义图标，让侧边栏更直观。

**为什么适合你：** 你的 vault 目录结构已经很完善（00_收件箱、10_日常、20_工作……），加上图标后视觉辨识度更高，找文件夹更快。

**效果：**
- 📥 00_收件箱
- 📅 10_日常
- 💼 20_工作
- 📚 30_学习
- 🎓 35_考研
- 🚀 40_创业

**上手难度极低：** 安装后右键文件夹 → Change icon，选一个喜欢的就行。

---

### 6. Banners

**插件 ID：** `obsidian-banners`

**是什么：** 在笔记顶部添加横幅图片，类似 Notion 的封面图效果。

**为什么适合你：** 你的 MOC 文件（`_MOC.md`）和重要项目文档加上封面图后，视觉效果会好很多，打开笔记时有仪式感。

**用法：** 在 frontmatter 里加一行：
```yaml
banner: "https://images.unsplash.com/photo-xxx"
```
或者拖拽本地图片到笔记顶部。

**注意：** 纯颜值插件，不影响功能，按需安装。

---

### 7. Kanban ⭐ 推荐

**插件 ID：** `obsidian-kanban`

**是什么：** 在 Obsidian 里创建看板（类似 Trello），每张卡片可以链接到一篇笔记。

**为什么适合你：** 你的 `40_创业/小红书/` 有草稿箱和已发布两个状态，用看板管理内容发布流程非常直观：

```
待写 → 草稿中 → 待发布 → 已发布
```

也可以用来管理考研复习进度：
```
待学 → 学习中 → 待复习 → 已掌握
```

---

### 8. Typewriter Mode

**插件 ID：** `obsidian-typewriter-mode`

**是什么：** 写作时光标始终保持在屏幕中央，上方是已写内容，下方是空白，像打字机一样。同时可以高亮当前行，让注意力集中在正在写的句子上。

**为什么适合你：** 写日记、写公众号文章时，这个模式能减少分心，写作体验更沉浸。

**功能：**
- 光标居中滚动
- 当前行高亮
- 其他行文字变暗（Focus Mode）
- 可以按快捷键随时开关

---

### 9. Minimal Theme Settings ⭐ 推荐

**插件 ID：** `obsidian-minimal-settings`

**是什么：** 你已经安装了 Minimal 主题，这个配套插件可以解锁更多定制选项：颜色、字体、布局宽度、图片样式、表格样式等，全部通过 UI 点选，不用写 CSS。

**为什么适合你：** Minimal 主题已经在你的 vault 里了，装这个插件相当于给它解锁高级设置面板，零成本提升颜值。

**推荐设置：**
- 开启 `Image grids`（图片自动排列成网格）
- 开启 `Trim Obsidian UI`（隐藏不常用的界面元素）
- 字体选 `LXGW WenKai`（霞鹜文楷，中文显示很好看）

---

### 10. Auto Link Title ⭐ 推荐

**插件 ID：** `obsidian-auto-link-title`

**是什么：** 粘贴 URL 时自动抓取网页标题，生成 `[标题](URL)` 格式的 Markdown 链接，而不是裸 URL。

**为什么适合你：** 你的笔记里有大量网页链接（学城文档、技术文章、InStreet 帖子），手动写标题很麻烦。装了这个插件，直接粘贴 URL，自动变成带标题的链接。

**效果对比：**
- 之前：`https://km.sankuai.com/page/123456`
- 之后：`[供应链实时数仓建设方案](https://km.sankuai.com/page/123456)`

**注意：** 内网链接（sankuai.com）可能抓不到标题，但外网链接完全没问题。

## 我的建议

根据你的使用场景，分优先级安装：

**第一批（立刻装，收益最高）**
- Tasks — 解决日记任务分散问题
- Linter — 自动规范 frontmatter 格式
- Auto Link Title — 粘贴链接更方便
- Minimal Theme Settings — 已有 Minimal 主题，零成本增强

**第二批（按需装）**
- Excalidraw — 如果你需要画架构图
- Kanban — 如果你想管理小红书内容发布流程
- QuickAdd — 如果你觉得新建笔记的流程还不够快

**第三批（颜值党）**
- Iconize — 文件夹图标
- Banners — 笔记封面
- Typewriter Mode — 沉浸写作
