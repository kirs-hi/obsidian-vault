# 综合案例 - 客户信息管理系统

## 需求说明

实现一个命令行客户信息管理系统，功能包括：

1. 显示主菜单
2. 添加客户
3. 删除客户
4. 修改客户信息
5. 查询客户（按姓名）
6. 显示所有客户
7. 退出系统

## 设计思路

### Customer 类

封装客户数据：

```python
class Customer:
    def __init__(self, cid, name, age, phone, email):
        self.cid = cid
        self.name = name
        self.age = age
        self.phone = phone
        self.email = email

    def __str__(self):
        return f"{self.cid}\t{self.name}\t{self.age}\t{self.phone}\t{self.email}"
```

### CustomerManager 类

管理客户列表，提供增删改查方法：

```python
class CustomerManager:
    def __init__(self):
        self.customers = []
        self._next_id = 1

    def add(self, name, age, phone, email):
        """添加客户"""
        customer = Customer(self._next_id, name, age, phone, email)
        self.customers.append(customer)
        self._next_id += 1
        print("添加成功！")

    def delete(self, cid):
        """根据 ID 删除客户"""
        for i, c in enumerate(self.customers):
            if c.cid == cid:
                self.customers.pop(i)
                print("删除成功！")
                return
        print("未找到该客户。")

    def update(self, cid, **kwargs):
        """根据 ID 修改客户信息"""
        for c in self.customers:
            if c.cid == cid:
                for key, value in kwargs.items():
                    if value:  # 只更新非空字段
                        setattr(c, key, value)
                print("修改成功！")
                return
        print("未找到该客户。")

    def find_by_name(self, name):
        """按姓名查询"""
        results = [c for c in self.customers if name in c.name]
        return results

    def show_all(self):
        """显示所有客户"""
        if not self.customers:
            print("暂无客户信息。")
            return
        print("ID\t姓名\t年龄\t电话\t\t邮箱")
        print("-" * 60)
        for c in self.customers:
            print(c)
```

## 主程序框架

```python
def main():
    manager = CustomerManager()

    while True:
        print("\n" + "=" * 40)
        print("      客户信息管理系统")
        print("=" * 40)
        print("  1. 添加客户")
        print("  2. 删除客户")
        print("  3. 修改客户")
        print("  4. 查询客户")
        print("  5. 显示所有")
        print("  6. 退出系统")
        print("=" * 40)

        choice = input("请选择操作(1-6): ").strip()

        if choice == "1":
            name = input("姓名: ").strip()
            age = input("年龄: ").strip()
            phone = input("电话: ").strip()
            email = input("邮箱: ").strip()
            try:
                age = int(age)
                manager.add(name, age, phone, email)
            except ValueError:
                print("年龄必须是数字！")

        elif choice == "2":
            try:
                cid = int(input("请输入要删除的客户ID: "))
                confirm = input(f"确认删除ID={cid}的客户？(y/n): ")
                if confirm.lower() == "y":
                    manager.delete(cid)
            except ValueError:
                print("ID必须是数字！")

        elif choice == "3":
            try:
                cid = int(input("请输入要修改的客户ID: "))
                print("(直接回车表示不修改)")
                name = input("新姓名: ").strip()
                age = input("新年龄: ").strip()
                phone = input("新电话: ").strip()
                email = input("新邮箱: ").strip()
                age = int(age) if age else None
                manager.update(cid, name=name or None,
                             age=age, phone=phone or None,
                             email=email or None)
            except ValueError:
                print("输入格式错误！")

        elif choice == "4":
            name = input("请输入要查询的姓名: ").strip()
            results = manager.find_by_name(name)
            if results:
                print(f"找到 {len(results)} 条记录:")
                for c in results:
                    print(c)
            else:
                print("未找到匹配的客户。")

        elif choice == "5":
            manager.show_all()

        elif choice == "6":
            confirm = input("确认退出？(y/n): ")
            if confirm.lower() == "y":
                print("感谢使用，再见！")
                break

        else:
            print("无效选择，请重新输入。")


if __name__ == "__main__":
    main()
```

## 核心知识点整合

这个案例综合运用了 Python 基础的多个核心知识点：

**类与对象**：Customer 类封装数据，CustomerManager 类封装业务逻辑，体现了面向对象的封装思想。`__str__` 方法自定义打印格式，`setattr` 动态设置属性。

**列表操作**：使用列表存储客户，涉及 append 添加、pop 删除、列表推导式过滤查询、enumerate 遍历获取索引。

**输入输出**：input 获取用户输入，strip 去除空白，格式化字符串输出，制表符对齐显示。

**异常处理**：try/except 捕获 ValueError，防止类型转换失败导致程序崩溃，保证程序健壮性。

**循环与条件**：while True 实现主循环，if/elif/else 分支处理不同操作，break 退出循环。
