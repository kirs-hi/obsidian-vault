# Python 流程控制语句

## 顺序结构

代码从上到下逐行执行，最基本的结构。

## 分支结构

### 单分支 if

```python
age = 18
if age >= 18:
    print("已成年")
```

### 双分支 if-else

```python
score = 75
if score >= 60:
    print("及格")
else:
    print("不及格")
```

### 多分支 if-elif-else

```python
score = 85
if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

### 嵌套分支

```python
sex = "男"
age = 20
if sex == "男":
    if age >= 18:
        print("成年男性")
    else:
        print("未成年男性")
else:
    print("女性")
```

### match-case（Python 3.10+）

```python
status = 404
match status:
    case 200:
        print("OK")
    case 404:
        print("Not Found")
    case 500:
        print("Server Error")
    case _:
        print("Unknown")
```

支持模式匹配：

```python
point = (0, 5)
match point:
    case (0, 0):
        print("原点")
    case (0, y):
        print(f"Y轴上，y={y}")
    case (x, 0):
        print(f"X轴上，x={x}")
    case (x, y):
        print(f"({x}, {y})")
```

### 三目运算符

```python
a, b = 10, 20
max_val = a if a > b else b  # 20

# 等价于
if a > b:
    max_val = a
else:
    max_val = b
```

## 循环结构

### while 循环

```python
# 计算 1+2+...+100
total = 0
i = 1
while i <= 100:
    total += i
    i += 1
print(total)  # 5050
```

### while-else

```python
i = 1
while i <= 5:
    print(i)
    i += 1
else:
    # 循环正常结束（非 break 退出）时执行
    print("循环正常结束")
```

### for 循环

```python
# 遍历序列
for char in "Python":
    print(char)

# 遍历列表
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
```

### for-else

```python
# 查找元素
nums = [1, 3, 5, 7, 9]
target = 4
for n in nums:
    if n == target:
        print("找到了")
        break
else:
    # 没有 break 时执行
    print("没找到")
```

### range()

```python
range(5)          # 0, 1, 2, 3, 4
range(1, 6)       # 1, 2, 3, 4, 5
range(0, 10, 2)   # 0, 2, 4, 6, 8
range(10, 0, -1)  # 10, 9, 8, ..., 1

# 常见用法
for i in range(5):
    print(i)
```

### 嵌套循环

```python
# 九九乘法表
for i in range(1, 10):
    for j in range(1, i + 1):
        print(f"{j}×{i}={i*j}", end="\t")
    print()
```

## 循环控制

### break — 跳出整个循环

```python
for i in range(10):
    if i == 5:
        break
    print(i)  # 0 1 2 3 4
```

### continue — 跳过本次迭代

```python
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)  # 1 3 5 7 9
```

### pass — 占位符（什么都不做）

```python
for i in range(10):
    pass  # TODO: 待实现

def my_func():
    pass  # 空函数占位
```

## 案例：兔子数量问题

> 第 1 周有 2 只兔子，以后每周增加上周的 2 倍，求第 N 周有多少只兔子。

```python
n = int(input("请输入周数："))
rabbits = 2

for week in range(2, n + 1):
    rabbits += rabbits * 2  # 增加上周的2倍，即变为原来的3倍

print(f"第{n}周有{rabbits}只兔子")

# 也可以理解为每周变为 3 倍：
# 第1周: 2
# 第2周: 2 + 2*2 = 6  (即 2*3)
# 第3周: 6 + 6*2 = 18 (即 6*3)
# 规律: rabbits = 2 * (3 ** (n-1))
```

等价的数学公式写法：

```python
n = int(input("请输入周数："))
rabbits = 2 * (3 ** (n - 1))
print(f"第{n}周有{rabbits}只兔子")
```

---

> [!tip] 选择循环的建议
> - 知道循环次数 → 用 `for`
> - 不知道循环次数、根据条件判断 → 用 `while`
> - `else` 子句适合"查找"场景：找到就 `break`，没找到走 `else`
