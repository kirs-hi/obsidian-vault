# Python 概述与环境搭建

## Python 起源

- 创始人：Guido van Rossum（龟叔）
- 诞生：1989 年圣诞假期开始编写，1991 年发布第一个版本
- 名字来源：英国喜剧团体 Monty Python
- 设计哲学：优雅、明确、简单（`import this` 查看 Python 之禅）

## Python 能做什么

| 领域 | 说明 |
|------|------|
| Web 开发 | Django、Flask、FastAPI |
| 网络爬虫 | Scrapy、Requests、BeautifulSoup |
| 数据分析 | Pandas、NumPy、Matplotlib |
| AI/机器学习 | TensorFlow、PyTorch、scikit-learn |
| 自动化运维 | Ansible、Fabric、脚本自动化 |
| 金融量化 | 量化交易策略、风险分析 |
| 游戏开发 | Pygame |
| 桌面应用 | PyQt、Tkinter |

## 优缺点

优点：语法简洁易学、第三方库极其丰富、跨平台可移植、可扩展（调用 C/C++）、开源社区活跃

缺点：运行速度相对较慢（解释型语言）、代码无法加密（发布即源码）、GIL 限制多线程

## Python 版本

- Python 2 已于 2020.1.1 正式停止维护
- 当前推荐使用 Python 3.12.8
- Python 2 和 3 不完全兼容，新项目一律用 Python 3

## Python 解释器

| 解释器 | 说明 |
|--------|------|
| CPython | 官方默认，C 语言实现，最常用 |
| IPython | 增强交互式，基于 CPython |
| PyPy | JIT 编译，运行速度快 |
| Jython | 运行在 JVM 上 |
| IronPython | 运行在 .NET 平台 |

## 安装 Python

1. 官网下载：https://www.python.org/downloads/
2. 选择 installer 安装包（不要选 embeddable）
3. 安装时勾选 Add Python to PATH
4. 验证安装：

```python
# 终端输入
python --version
# 或
python3 --version
```

## PyCharm IDE

创建项目：File - New Project - 选择解释器路径

常用设置：
- 主题：File - Settings - Appearance - Theme
- 字体：Settings - Editor - Font（推荐 JetBrains Mono, 16px）
- 快捷键方案：Settings - Keymap

## 第一个程序

### 交互式模式

```python
>>> print("Hello, Python!")
Hello, Python!
```

### 脚本模式

```python
# hello.py
print("Hello, Python!")
```

终端运行：`python hello.py`

### IDE 模式

在 PyCharm 中右键 - Run 即可执行。

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + / | 注释/取消注释 |
| Ctrl + D | 复制当前行 |
| Ctrl + Y | 删除当前行 |
| Ctrl + Alt + L | 格式化代码 |
| Shift + F6 | 重命名 |
| Ctrl + Shift + F | 全局搜索 |
| Alt + Enter | 快速修复 |
| Shift + Enter | 下方插入新行 |
| Double Shift | 全局搜索一切 |

---

> [!tip] 学习建议
> 初学阶段建议同时使用交互式模式（快速验证）和 IDE（写完整程序），培养编码手感。
