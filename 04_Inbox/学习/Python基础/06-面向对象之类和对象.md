# 面向对象之类和对象

## 面向过程 vs 面向对象

- **面向过程**：以步骤为核心，按顺序一步步执行（适合简单脚本）
- **面向对象**：以对象为核心，将数据和操作封装在一起（适合复杂系统）

## 类和对象的关系

- **类（Class）**：模板/蓝图，定义了属性（数据）和方法（行为）
- **对象（Object）**：类的实例，根据模板创建的具体个体

## 定义类

```python
class Student:
    # 类属性（所有实例共享）
    school = "尚硅谷"

    # 初始化方法
    def __init__(self, name, age):
        # 实例属性（每个实例独有）
        self.name = name
        self.age = age

    # 实例方法
    def study(self):
        print(f"{self.name}正在学习")
```

## 类的操作

```python
# 成员引用：通过类名访问类属性
print(Student.school)  # 尚硅谷

# 实例化：创建对象
stu = Student("Alice", 20)
print(stu.name)  # Alice
stu.study()      # Alice正在学习
```

## `__init__()` 方法

构造初始化方法，创建对象时自动调用，用于设置实例的初始状态：

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

p = Person("Bob", 25)  # 自动调用 __init__
```

## self

`self` 代表实例本身，通过它访问实例的属性和方法。调用时不需要手动传入，Python 自动传递。

```python
class Cat:
    def __init__(self, name):
        self.name = name  # self.name 是实例属性

    def say(self):
        print(f"我是{self.name}")  # 通过 self 访问
```

## 属性：类属性 vs 实例属性

```python
class Dog:
    count = 0  # 类属性，所有实例共享

    def __init__(self, name):
        self.name = name   # 实例属性，每个实例独有
        Dog.count += 1

d1 = Dog("旺财")
d2 = Dog("来福")
print(Dog.count)  # 2
```

- 类属性通过 `类名.属性` 访问和修改
- 实例属性通过 `实例.属性` 访问和修改
- 实例可以读取类属性，但赋值会创建同名实例属性（遮蔽类属性）

## 方法类型

### 实例方法

第一个参数为 `self`，通过实例调用：

```python
class MyClass:
    def instance_method(self):
        print(f"实例方法，访问实例：{self}")
```

### 类方法

用 `@classmethod` 装饰，第一个参数为 `cls`（代表类本身）：

```python
class MyClass:
    count = 0

    @classmethod
    def get_count(cls):
        return cls.count
```

### 静态方法

用 `@staticmethod` 装饰，不需要 `self` 或 `cls`，与类相关但不依赖实例或类状态：

```python
class MathUtils:
    @staticmethod
    def add(a, b):
        return a + b
```

## 特殊方法（魔术方法）

```python
class Person:
    def __new__(cls, *args, **kwargs):
        """创建实例（在 __init__ 之前调用）"""
        return super().__new__(cls)

    def __init__(self, name):
        """初始化实例"""
        self.name = name

    def __del__(self):
        """析构方法，对象被销毁时调用"""
        print(f"{self.name}被销毁")

    def __str__(self):
        """print() 时的显示，面向用户"""
        return f"Person({self.name})"

    def __repr__(self):
        """交互式环境的显示，面向开发者"""
        return f"Person(name='{self.name}')"

    def __getattribute__(self, name):
        """访问任何属性时触发"""
        return super().__getattribute__(name)
```

## 动态添加属性与方法

```python
import types

class MyClass:
    pass

obj = MyClass()

# 动态添加属性
obj.name = "动态属性"

# 动态添加方法
def hello(self):
    print(f"hello, {self.name}")

obj.hello = types.MethodType(hello, obj)
obj.hello()  # hello, 动态属性
```

## `__slots__` 限制实例属性

```python
class Point:
    __slots__ = ('x', 'y')  # 只允许 x, y 属性

    def __init__(self, x, y):
        self.x = x
        self.y = y

p = Point(1, 2)
p.z = 3  # AttributeError: 'Point' object has no attribute 'z'
```

`__slots__` 的优点：节省内存（不再使用 `__dict__`），提高属性访问速度。
