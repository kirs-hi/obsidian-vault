# 函数

## 函数概念与定义

函数是组织好的、可重复使用的代码块，用于实现单一或相关联的功能。

```python
def 函数名(参数):
    """函数说明文档"""
    函数体
    return 返回值
```

**命名规则**：字母/下划线开头，只能包含字母、数字、下划线，不能使用关键字，建议使用 `snake_case` 风格。

## 形参和实参

- **形参**：定义函数时括号中的变量，是占位符
- **实参**：调用函数时传入的实际值

## 参数类型

### 必须参数（位置参数）

按顺序传递，数量必须匹配：

```python
def greet(name, msg):
    print(f"{name}, {msg}")

greet("Alice", "早上好")
```

### 关键字参数

通过参数名传值，不依赖顺序：

```python
greet(msg="你好", name="Bob")
```

### 默认值参数

给参数设定默认值，调用时可省略：

```python
def greet(name, msg="你好"):
    print(f"{name}, {msg}")

greet("Alice")  # Alice, 你好
```

### 不定长参数

```python
# *args：接收任意数量位置参数，打包为元组
def func(*args):
    print(args)  # (1, 2, 3)

# **kwargs：接收任意数量关键字参数，打包为字典
def func(**kwargs):
    print(kwargs)  # {'a': 1, 'b': 2}

# 混合使用
def func(*args, **kwargs):
    pass
```

### 解包传参

```python
lst = [1, 2, 3]
func(*lst)  # 等价于 func(1, 2, 3)

d = {'name': 'Alice', 'age': 20}
func(**d)   # 等价于 func(name='Alice', age=20)
```

### 强制位置参数和关键字参数

```python
def func(a, b, /, c, *, d, e):
    pass
# a, b 只能位置传参（/之前）
# d, e 只能关键字传参（*之后）
# c 两种方式都行
```

## 参数传递机制

Python 中一切皆对象，变量是对象的引用（标签）。

- **不可变类型**（int/str/tuple）：函数内重新赋值不影响外部，相当于传值
- **可变类型**（list/dict/set）：函数内修改会影响外部原对象，相当于传引用

```python
def modify(lst, num):
    lst.append(4)   # 修改原列表
    num = num + 1   # 不影响外部 num

my_list = [1, 2, 3]
my_num = 10
modify(my_list, my_num)
print(my_list)  # [1, 2, 3, 4]
print(my_num)   # 10
```

### `*=` 与 `= ... *` 的区别

```python
# 对于可变类型（如 list）
var1 = [1, 2]
var1 *= 2        # 就地修改，var1 还是原对象 → [1, 2, 1, 2]

var1 = [1, 2]
var1 = var1 * 2  # 创建新对象，var1 指向新列表
```

### 防止函数修改列表

传入切片副本：

```python
func(my_list[:])  # 传副本，原列表不受影响
```

## 函数说明文档

```python
def add(a, b):
    """
    计算两个数的和

    参数:
        a: 第一个数
        b: 第二个数
    返回:
        两数之和
    """
    return a + b

# 查看文档
help(add)
print(add.__doc__)
```

## 返回值

函数可以返回多个值，本质是返回元组：

```python
def min_max(lst):
    return min(lst), max(lst)

lo, hi = min_max([3, 1, 4, 1, 5])
```

没有 return 或 return 后无值，默认返回 `None`。

## 函数嵌套调用

```python
def inner():
    print("inner")

def outer():
    print("outer")
    inner()  # 在函数内部调用另一个函数

outer()
```

## 变量作用域

- **局部变量**：函数内部定义，函数外不可访问
- **全局变量**：函数外部定义，所有函数可读取

```python
x = 10  # 全局变量

def func():
    global x    # 声明使用全局变量
    x = 20      # 修改全局变量

def outer():
    y = 5
    def inner():
        nonlocal y  # 声明使用外层函数变量
        y = 10
    inner()
    print(y)  # 10
```

## 递归

函数调用自身，必须有基线条件（终止条件）和递归条件。

```python
def factorial(n):
    """计算阶乘"""
    if n == 1:       # 基线条件
        return 1
    return n * factorial(n - 1)  # 递归条件

print(factorial(5))  # 120
```

## 匿名函数 lambda

一行定义的简单函数：

```python
# 语法：lambda 参数: 表达式
add = lambda x, y: x + y
print(add(3, 4))  # 7
```

### lambda 配合内置函数

```python
# sorted 自定义排序
students = [('Alice', 90), ('Bob', 80), ('Cindy', 95)]
sorted(students, key=lambda s: s[1], reverse=True)

# map 映射
list(map(lambda x: x ** 2, [1, 2, 3]))  # [1, 4, 9]

# filter 过滤
list(filter(lambda x: x > 2, [1, 2, 3, 4]))  # [3, 4]

# reduce 累积（需导入）
from functools import reduce
reduce(lambda x, y: x + y, [1, 2, 3, 4])  # 10
```

## 函数注释（类型提示）

def greet(name: str, times: int = 1) -> str:
    return name * times
