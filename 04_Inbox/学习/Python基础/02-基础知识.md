# Python 基础知识

## 注释

```python
# 单行注释

'''
多行注释（单引号）
可以跨多行
'''

"""
多行注释（双引号）
也可以跨多行
"""
```

## 变量

Python 是**动态类型语言**，变量无需声明类型，赋值即创建：

```python
name = "Python"    # str
age = 35           # int
pi = 3.14          # float
is_easy = True     # bool
```

**命名规则：**
- 由字母、数字、下划线组成
- 不能以数字开头
- 区分大小写
- 不能使用关键字

**命名方法：**

```python
# 驼峰命名法（类名推荐）
MyClassName = "demo"

# 下划线命名法（变量/函数推荐）
my_variable = "demo"
```

## 关键字

```python
import keyword
print(keyword.kwlist)
# False, None, True, and, as, assert, async, await,
# break, class, continue, def, del, elif, else, except,
# finally, for, from, global, if, import, in, is, lambda,
# nonlocal, not, or, pass, raise, return, try, while, with, yield
```

## 常量

Python 没有 `const` 关键字，约定**全大写**表示常量：

```python
MAX_SIZE = 100
PI = 3.14159265
BASE_URL = "https://api.example.com"
```

## 进制

```python
a = 0b1010    # 二进制，值为 10
b = 0o17      # 八进制，值为 15
c = 0xFF      # 十六进制，值为 255

# 转换函数
bin(10)       # '0b1010'
oct(10)       # '0o12'
hex(255)      # '0xff'
int('0b1010', 2)  # 10
```

## 原码、反码、补码

| 类型 | 正数 | 负数 |
|------|------|------|
| 原码 | 本身 | 符号位为1，其余为值 |
| 反码 | 同原码 | 符号位不变，其余取反 |
| 补码 | 同原码 | 反码 + 1 |

正数三码合一。计算机内部用**补码**存储和运算。

```
+7: 原码 0000 0111  反码 0000 0111  补码 0000 0111
-7: 原码 1000 0111  反码 1111 1000  补码 1111 1001
```

## 数据类型

### int 整型

```python
x = 100
y = -50
big = 999999999999999999  # 支持任意大整数
```

**小整数池：** Python 预缓存 `[-5, 256]` 范围内的整数对象，这个范围内的相同值是同一对象：

```python
a = 256
b = 256
print(a is b)  # True

a = 257
b = 257
print(a is b)  # False（交互模式下）
```

### float 浮点型

```python
f = 3.14
e = 2.5e10  # 科学计数法

# 精度问题
print(0.1 + 0.2)  # 0.30000000000000004

# 解决方案：Decimal
from decimal import Decimal
print(Decimal('0.1') + Decimal('0.2'))  # 0.3
```

### bool 布尔型

```python
# bool 是 int 的子类
print(True + 1)    # 2
print(False + 1)   # 1
print(isinstance(True, int))  # True

# 以下值为 False：
# 0, 0.0, '', [], (), {}, set(), None
```

### str 字符串

```python
s1 = '单引号'
s2 = "双引号"
s3 = '''三引号
可以换行'''
s4 = r"原始字符串\n不转义"  # \n 不会换行
```

**intern 机制 / 字符串缓冲池：** 短字符串（仅含字母数字下划线）会被缓存复用。

### None

```python
x = None
print(type(x))    # <class 'NoneType'>
print(x is None)  # True（判断 None 用 is）
```

## 类型转换

### 隐式转换

```python
result = 1 + 2.0   # int + float → float (3.0)
result = True + 3  # bool + int → int (4)
```

### 显式转换

```python
int("123")        # 123
int(3.9)          # 3（截断，非四舍五入）
float("3.14")     # 3.14
str(100)          # "100"
bool(0)           # False
bool("hello")     # True
list((1,2,3))     # [1, 2, 3]
tuple([1,2,3])    # (1, 2, 3)
```

## 编码解码

```python
# 字符串 → 字节序列
s = "你好Python"
b = s.encode('utf-8')     # b'\xe4\xbd\xa0\xe5\xa5\xbdPython'

# 字节序列 → 字符串
s2 = b.decode('utf-8')    # '你好Python'

# 一个中文 UTF-8 占 3 字节，GBK 占 2 字节
```

## 输入输出

### input()

```python
name = input("请输入姓名：")  # 返回值始终是 str
age = int(input("请输入年龄："))  # 需手动转换类型
```

### print()

```python
print("Hello", "World", sep=", ", end="!\n")
# 输出：Hello, World!
```

### 格式化输出

```python
name = "Alice"
age = 25
score = 95.5

# 1. % 格式化
print("姓名：%s，年龄：%d，成绩：%.1f" % (name, age, score))

# 2. format()
print("姓名：{}，年龄：{}".format(name, age))
print("{0} is {1} years old".format(name, age))

# 3. f-string（推荐）
print(f"姓名：{name}，年龄：{age}，成绩：{score:.1f}")
print(f"{age = }")  # age = 25（调试技巧）
```

## 运算符

### 算术运算符

```python
10 / 3     # 3.333...（真除法）
10 // 3    # 3（整除）
10 % 3     # 1（取余）
2 ** 10    # 1024（幂运算）
```

### 赋值运算符

```python
a = 10
a += 5     # a = 15
a -= 3     # a = 12
a *= 2     # a = 24
a //= 5    # a = 4
# 注意：Python 没有 ++ 和 --
```

### 比较运算符

```python
# == 比较值，is 比较地址（是否同一对象）
a = [1, 2]
b = [1, 2]
print(a == b)   # True
print(a is b)   # False
```

### 逻辑运算符

```python
# and: 短路与，第一个 False 就返回
# or:  短路或，第一个 True 就返回
# not: 取反

print(1 and 2)      # 2
print(0 and 2)      # 0
print(1 or 2)       # 1
print(0 or 2)       # 2
```

### 位运算符

```python
a = 0b1010  # 10
b = 0b1100  # 12

a & b    # 0b1000 = 8  （按位与）
a | b    # 0b1110 = 14 （按位或）
a ^ b    # 0b0110 = 6  （按位异或）
~a       # -11         （按位取反）
a << 1   # 20          （左移，×2）
a >> 1   # 5           （右移，÷2）
```

### 成员 & 身份运算符

```python
# in / not in
print('a' in 'abc')        # True
print(1 in [1, 2, 3])     # True

# is / is not（判断是否同一对象）
print(None is None)        # True
```

### 运算符优先级（从高到低）

`**` → `~ + -`(单目) → `* / // %` → `+ -` → `>> <<` → `&` → `^ |` → `<= < > >=` → `== !=` → `is / in` → `not` → `and` → `or`

> [!tip] 记不住就加括号，可读性更重要。

## 编码规范

- 缩进统一用 **4 个空格**（不用 Tab）
- 每行不超过 **80** 个字符
- 顶层函数/类之间空 **2 行**，类内方法之间空 **1 行**
- 导入分组：标准库 → 第三方库 → 本地模块，组间空一行
- 变量/函数名用 `snake_case`，类名用 `PascalCase`，常量用 `ALL_CAPS`
