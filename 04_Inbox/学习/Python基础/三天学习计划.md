# Python 三天速学计划

## Day 1：基础语法 + 流程控制 + 容器（8-10h）

### 上午（3h）概述 + 快速入门 + 基础知识
- 快速扫读计算机基础、Python 简介（已有基础可跳）
- 重点：变量无需声明类型、小整数池[-5,256]、intern 机制
- 重点：f-string 格式化、海象运算符 `:=`、位运算补码逻辑

### 下午（3h）流程控制语句
- Python 特色：缩进即代码块
- match-case（Python 3.10+）
- for-else / while-else 语法
- range() 用法，每个案例动手敲

### 晚上（3h）容器数据类型
- 列表推导式（最核心）
- 字典遍历与操作
- 集合运算
- zip() 函数、切片操作
- **这是后续所有章节的基础，必须吃透**

---

## Day 2：函数 + 面向对象 + 文件操作（8-10h）

### 上午（3h）函数
- 可变/不可变参数传递机制
- `*args / **kwargs` 不定长参数、解包传参
- 闭包、装饰器概念
- lambda 配合 sorted/map/filter/reduce

### 下午（4h）面向对象
- 类的定义和实例化、`__init__` / `__str__`
- 继承和方法重写
- property 装饰器
- 封装（私有化）、多态
- 愤怒的小鸟案例：了解设计思路即可

### 晚上（2h）文件操作 + 异常处理
- with open 语法、读写模式
- os.walk 递归遍历
- try-except-else-finally
- 自定义异常、异常传递

---

## Day 3：模块/包 + 高级语法 + 进程线程 + 网络/正则（8-10h）

### 上午（3h）模块与包 + 高级语法
- import 机制、`__name__ == "__main__"`
- pip 管理、`__all__` 控制导出
- 深拷贝/浅拷贝
- 迭代器/生成器（yield）
- 装饰器深入（类装饰器、多层、带参）

### 下午（3h）进程与线程
- GIL 的影响
- 多进程 vs 多线程适用场景
- 进程池/线程池
- 互斥锁、进程间通信

### 晚上（2-3h）网络编程 + 正则表达式
- TCP/UDP 区别、HTTP 基础
- re 模块：search/findall/sub/split
- 常用匹配模式
- 综合案例：客户信息管理系统（有时间就写，没时间读懂）

## 效率提升技巧

- **已知映射法**：列表≈ArrayList，字典≈HashMap，元组≈不可变数组，生成器≈惰性求值，装饰器≈AOP
- 每天结束花 20 分钟回顾，用费曼技巧复述关键概念
- 有编程基础的重点关注 Python 独特之处，已知内容快速扫过
