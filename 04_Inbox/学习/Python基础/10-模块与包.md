# 模块与包

## 模块概述

一个 `.py` 文件就是一个模块（Module）。模块让我们能把代码组织成逻辑单元，实现代码复用和命名空间隔离。

## 创建模块

直接创建一个 `.py` 文件即可，文件名就是模块名（不含 `.py` 后缀）。

```python
# my_module.py
def greet(name):
    return f"Hello, {name}!"

def add(a, b):
    return a + b
```

## 导入模块

### import 模块名

导入整个模块，通过 `模块名.函数名` 调用：

```python
import my_module

result = my_module.add(1, 2)
print(my_module.greet("Python"))
```

可以用 `as` 起别名：

```python
import my_module as mm
mm.greet("World")
```

### from 模块名 import 函数名

局部导入，直接使用函数名：

```python
from my_module import greet, add

print(greet("Python"))
print(add(3, 4))
```

### from 模块名 import *

导入模块中所有公开成员，受 `__all__` 控制：

```python
from my_module import *
```

> 不推荐在大型项目中使用，容易造成命名冲突。

## 模块搜索顺序

Python 查找模块的顺序：

1. **当前目录**
2. **内置模块**（如 sys、os）
3. **sys.path** 中的路径（包含环境变量 PYTHONPATH 和安装路径）

```python
import sys
print(sys.path)  # 查看搜索路径
```

## `__all__`

控制 `from xxx import *` 时导出哪些成员：

```python
# my_module.py
__all__ = ["greet"]  # 只有 greet 会被 * 导入

def greet(name):
    return f"Hello, {name}!"

def _private():
    pass  # 不会被导出
```

## `__name__`

每个模块都有 `__name__` 属性：
- 模块被**直接运行**时，`__name__` 为 `"__main__"`
- 模块被**导入**时，`__name__` 为模块名

```python
# my_module.py
def main():
    print("测试代码")

if __name__ == "__main__":
    main()  # 只有直接运行此文件时才执行
```

这是防止测试代码在被导入时执行的标准做法。

## dir() 函数

查看模块内的所有名称：

```python
import math
print(dir(math))  # 列出 math 模块的所有属性和方法
```

## 包（Package）

### 包的概念

包是一个包含 `__init__.py` 文件的目录，用于组织多个模块。

```
my_package/
├── __init__.py      # 标识这是一个包
├── module_a.py
├── module_b.py
└── sub_package/
    ├── __init__.py
    └── module_c.py
```

### 创建包

`__init__.py` 可以为空，也可以定义包的初始化代码：

```python
# my_package/__init__.py
from .module_a import func_a
from .module_b import func_b
```

### 导入包的方式

```python
# 方式1：导入包中的模块
import my_package.module_a
my_package.module_a.func_a()

# 方式2：从包中导入模块
from my_package import module_a
module_a.func_a()

# 方式3：从包的模块中导入函数
from my_package.module_a import func_a
func_a()

# 方式4：导入子包
from my_package.sub_package import module_c
```

## 常用标准库

| 模块 | 用途 |
|------|------|
| `os` | 操作系统接口（文件/目录操作） |
| `sys` | 系统相关（命令行参数、路径） |
| `math` | 数学函数 |
| `random` | 随机数生成 |
| `datetime` | 日期时间处理 |
| `json` | JSON 编解码 |
| `re` | [[14-正则表达式|正则表达式]] |
| `collections` | 高级数据结构（Counter, defaultdict 等） |

```python
import os
print(os.getcwd())  # 当前工作目录

import random
print(random.randint(1, 100))  # 1-100随机整数

from datetime import datetime
print(datetime.now())  # 当前时间
```

## 引入第三方库

### pip 安装

```bash
pip install requests        # 安装
pip install requests==2.28  # 指定版本
pip list                    # 查看已安装
pip uninstall requests      # 卸载

# 使用国内镜像加速
pip install requests -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### PyCharm 中安装

Settings → Project → Python Interpreter → 点击 `+` 号搜索安装。

## 打包自己的库

使用 `setuptools` 打包：

```python
# setup.py
from setuptools import setup, find_packages

setup(
    name="my_package",
    version="1.0.0",
    packages=find_packages(),
    install_requires=["requests>=2.20"],
    author="Your Name",
    description="A sample package",
)
```

```bash
# 打包
python setup.py sdist bdist_wheel

# 安装本地包
pip install .

# 上传到 PyPI
pip install twine
twine upload dist/*
```

---

> [!tip] 核心要点
> - 模块 = `.py` 文件，包 = 含 `__init__.py` 的目录
> - `if __name__ == "__main__":` 是模块的标准入口守卫
> - 优先使用 `from xxx import yyy` 明确导入
