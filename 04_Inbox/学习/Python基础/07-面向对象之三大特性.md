# 面向对象之三大特性

## 封装

将数据和操作封装在类内部，对外隐藏实现细节，通过接口访问。

### 私有化

```python
class Account:
    def __init__(self, name, balance):
        self._name = name        # 单下划线：约定非公开，外部仍可访问
        self.__balance = balance  # 双下划线：名称改写，外部无法直接访问

a = Account("Alice", 1000)
print(a._name)       # Alice（可以访问，但不建议）
# print(a.__balance) # AttributeError
print(a._Account__balance)  # 1000（通过 _类名__属性 强制访问）
```

### 私有属性和私有方法

```python
class Phone:
    def __init__(self):
        self.__voltage = 3.7  # 私有属性

    def __check_voltage(self):  # 私有方法
        return self.__voltage > 3.0

    def call(self):  # 公开方法，内部使用私有成员
        if self.__check_voltage():
            print("打电话")
```

### property 装饰器

将方法伪装成属性访问，实现读写控制：

```python
class Circle:
    def __init__(self, radius):
        self.__radius = radius

    @property
    def radius(self):
        """只读属性 / getter"""
        return self.__radius

    @radius.setter
    def radius(self, value):
        """setter，可加验证逻辑"""
        if value < 0:
            raise ValueError("半径不能为负")
        self.__radius = value

    @property
    def area(self):
        """只读计算属性"""
        return 3.14159 * self.__radius ** 2

c = Circle(5)
print(c.radius)  # 5（像属性一样访问）
c.radius = 10    # 调用 setter
print(c.area)    # 314.159
# c.area = 100   # AttributeError（只读）
```

## 继承

子类复用父类的属性和方法，并可扩展或修改。

### 单继承

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print(f"{self.name}发出声音")

class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name)  # 调用父类 __init__
        self.breed = breed

    def fetch(self):
        print(f"{self.name}去捡球")

dog = Dog("旺财", "柴犬")
dog.speak()  # 旺财发出声音（继承自父类）
dog.fetch()  # 旺财去捡球（子类自有）
```

### 多继承

```python
class Flyable:
    def fly(self):
        print("飞行中")

class Swimmable:
    def swim(self):
        print("游泳中")

class Duck(Flyable, Swimmable):
    pass

duck = Duck()
duck.fly()   # 飞行中
duck.swim()  # 游泳中
```

### 复用父类方法

```python
class Child(Parent):
    def method(self):
        # 方式1：super()（推荐）
        super().method()

        # 方式2：显式指定父类名
        Parent.method(self)
```

### 方法解析顺序 MRO

多继承时，Python 使用 C3 线性化算法确定方法查找顺序：

```python
class A:
    pass

class B(A):
    pass

class C(A):
    pass

class D(B, C):
    pass

print(D.mro())
# [D, B, C, A, object]
```

### 方法重写

子类定义与父类同名的方法，调用时优先使用子类版本：

```python
class Animal:
    def speak(self):
        print("...")

class Cat(Animal):
    def speak(self):  # 重写父类方法
        print("喵喵喵")

cat = Cat()
cat.speak()  # 喵喵喵
```

## 多态

同一方法在不同对象上表现出不同行为。Python 天然支持多态（鸭子类型：不关心类型，只关心是否有对应方法）。

```python
class Dog:
    def speak(self):
        return "汪汪"

class Cat:
    def speak(self):
        return "喵喵"

class Duck:
    def speak(self):
        return "嘎嘎"

# 不关心具体类型，只要有 speak 方法就行
def animal_speak(animal):
    print(animal.speak())

animal_speak(Dog())   # 汪汪
animal_speak(Cat())   # 喵喵
animal_speak(Duck())  # 嘎嘎
```

鸭子类型的核心思想：「如果它走路像鸭子，叫声像鸭子，那它就是鸭子。」不需要继承同一个基类，只要实现了相同的方法接口即可。
