# 文件操作

## 文件分类

- **纯文本文件**：以字符编码存储（如 UTF-8），可直接阅读（.txt, .py, .csv）
- **二进制文件**：以字节存储，需要专用程序打开（.jpg, .mp3, .exe）

## 文件路径

- **相对路径**：相对于当前工作目录，如 `data/file.txt`
- **绝对路径**：从根目录开始的完整路径，如 `/Users/name/data/file.txt`

## 打开文件

```python
file = open(file, mode='r', encoding='utf-8')
```

### 常用模式

| 模式 | 说明 |
|------|------|
| `r` | 只读（默认），文件不存在则报错 |
| `w` | 只写，文件存在则覆盖，不存在则创建 |
| `a` | 追加，在末尾写入，不存在则创建 |
| `rb` | 二进制读 |
| `wb` | 二进制写 |
| `r+` | 读写，文件不存在则报错 |

## 关闭文件

```python
file = open("test.txt", "r")
# ... 操作文件 ...
file.close()  # 必须关闭，释放资源
```

## with 语法（推荐）

自动管理文件关闭，即使发生异常也能正确关闭：

```python
with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()
# 离开 with 块后自动关闭
```

## 写数据

```python
# write：写入字符串
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("第一行\n")
    f.write("第二行\n")

# writelines：写入字符串列表（不自动加换行）
lines = ["aaa\n", "bbb\n", "ccc\n"]
with open("output.txt", "w", encoding="utf-8") as f:
    f.writelines(lines)
```

## 读数据

```python
with open("test.txt", "r", encoding="utf-8") as f:
    # read()：读取全部内容
    content = f.read()

    # read(n)：读取 n 个字符
    chunk = f.read(10)

    # readline()：读取一行
    line = f.readline()

    # readlines()：读取所有行，返回列表
    lines = f.readlines()

# 逐行读取（推荐，内存友好）
with open("test.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())
```

## 常用函数：os.walk() 递归遍历目录

```python
import os

for dirpath, dirnames, filenames in os.walk("/path/to/dir"):
    for filename in filenames:
        filepath = os.path.join(dirpath, filename)
        print(filepath)
```

- `dirpath`：当前遍历到的目录路径
- `dirnames`：当前目录下的子目录列表
- `filenames`：当前目录下的文件列表

## 案例：文件拷贝

```python
def copy_file(src, dst):
    """拷贝文件（支持大文件，二进制模式）"""
    with open(src, "rb") as f_src:
        with open(dst, "wb") as f_dst:
            while True:
                chunk = f_src.read(1024)  # 每次读 1KB
                if not chunk:
                    break
                f_dst.write(chunk)

copy_file("source.jpg", "backup.jpg")
```
