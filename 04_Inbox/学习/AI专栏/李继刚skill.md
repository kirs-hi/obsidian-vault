## ljg-skills 技能集（19个技能）

### 阅读与研究类

| 技能名 | 功能 |
|--------|------|
| **ljg-paper** | 论文阅读器，为非学术人士提取论文核心想法，输出 org-mode 笔记 |
| **ljg-paper-river** | 论文溯源，倒读法递归挖前序论文（最多5层），呈现问题演化脉络 |
| **ljg-read** | 伴读，英文三层翻译（信达雅）+ 结构标注 + 深度提问 + 跨领域旁逸 |

### 写作与表达类

| 技能名            | 功能                                 |
| -------------- | ---------------------------------- |
| **ljg-writes** | 写作引擎，"三刀写作法"，像手术刀剖开一个观点，1000-1500字 |
| **ljg-plain**  | 白话引擎，把任何内容改写到聪明的十二岁小孩也能懂           |
| **ljg-qa**     | 信息提问机，把文章/论文/书的核心观点抽成 Q-A 链        |

### 概念解剖与思维类

| 技能名 | 功能 |
|--------|------|
| **ljg-learn** | 概念解剖，从八个方向切开一个概念（历史、辩证、现象、语言、形式、存在、美感、元反思） |
| **ljg-think** | 追本之箭，给一个观点或现象，纵向深钻到不可再分的本质 |
| **ljg-rank** | 降秩引擎，给一个领域，找出背后不可再少的独立生成器 |

### 分析类

| 技能名 | 功能 |
|--------|------|
| **ljg-invest** | 投资分析，核心判断项目是否是一台"秩序创造机器" |
| **ljg-relationship** | 关系分析，五层结构诊断 + 精神分析 |

### 表达与传播类

| 技能名 | 功能 |
|--------|------|
| **ljg-card** | 内容铸卡，将内容转为 PNG 视觉卡片（长图、信息图、海报、漫画、白板、碑刻等7种模具） |
| **ljg-present** | 演讲铸造器，基于 orgmode outline 层级 1:1 视觉化呈现 |
| **ljg-roundtable** | 圆桌讨论，结构化多人辩证对话框架 |

### 辅助工具类

| 技能名 | 功能 |
|--------|------|
| **ljg-word** | 单词精通，深度拆解一个英语单词的核心语义 |
| **ljg-travel** | 旅行研究，输入城市名，生成深度文化研究文档 + 便携卡片 |
| **ljg-skill-map** | 技能地图，扫描所有已安装技能，渲染可视化总览 |

## 设计哲学

他的 Skill 有几个显著特征：使用 Lisp（S-expression）语法极致压缩表达；强调"工具即认知"；每个技能只解决一类问题但解决得很深；反 AI 生成痕迹（禁 Inter 字体、禁纯黑、禁 AI 文案腔）；输出物优先（PNG/文章/卡片）。
