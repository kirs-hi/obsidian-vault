# 考研刷题产品调研 — 研可岸真题网 & CodeBrick 码砖

## 一、研可岸真题网（zhenti.kaoyansou.cn）

### 1.1 产品概述

**研可岸真题网**是「研可岸」品牌旗下的考研公共课真题在线训练平台，专注于将历年考研真题按考点维度拆解重组，辅以统计分析数据（考频、分值占比、题型分布等），帮助考生高效定位高频考点。

- 网址：zhenti.kaoyansou.cn（子域名形式，/cse408 为408计算机统考专区）
- 母品牌：研可岸（yankean.cn），浙ICP备2023002559号
- 创始人：B站UP主「学长小池」（UID 298473865）
- 团队：全部由研究生组成
- 科目覆盖：数学一/二/三、408计算机统考、考研英语

### 1.2 核心功能

1. **考点维度汇总**：将真题按知识点体系拆解，形成考点树结构（数学覆盖1987-2026年）
2. **考频统计分析**：每个考点展示"历史总计题数"、"出现率"、"每年平均考察分值"、"题型分布"
3. **历年考察频率分布图**：可视化逐年出题频率变化
4. **多维排序**：按出现年份数/平均分值/总题数排序，快速锁定重点
5. **组合刷题**：按考点聚合真题，按题型筛选进入刷题模式
6. **年份汇总**：以年份为维度查看整套试卷

### 1.3 商业模式

**免费增值（Freemium）**：

| 模块 | 定价 |
|------|------|
| 数学/408/英语真题网 | 免费 |
| 择校网会员 | 拼团¥1（原价¥29.9） |
| 调剂小程序会员 | ¥88 |
| 1V1择校辅导 | 付费（市场价1/4~1/6） |

**变现路径**：免费真题工具获客 → 公众号沉淀（登录必须关注公众号获取登录码）→ 向择校/调剂/辅导付费服务转化。

### 1.4 技术栈

- 前端：Vue.js + Element Plus（SPA）
- 静态资源：阿里云 OSS（cn-beijing）
- 后端推测：Node.js 或 Java（Spring Boot）+ MySQL
- 支付：有赞（youzan.com）SaaS
- 登录：微信公众号数字码机制（非传统账密）

### 1.5 用户量与知名度

| 指标 | 数据 |
|------|------|
| 全产品总使用人数 | ~15万（商协汇数据） |
| B站爆款视频播放 | 《我做了个很牛的考研择校网站》251.6万播放 |
| 抖音"学长小池" | 粉丝 3.7-4万，获赞 30万 |
| 微信公众号 | 核心私域入口，预估关注数万 |

### 1.6 优缺点

**优点**：
- 差异化强：不是堆PDF，做了"考点拆解+统计分析"的数据加工层
- 数据全：数学追溯到1987年，跨度近40年
- 免费策略得当：核心功能免费，降低使用门槛
- 创始人IP加持：B站百万播放爆款
- 产品生态协同：真题→择校→调剂→辅导，漏斗清晰
- 定价亲民：贴合学生消费能力

**缺点**：
- 技术壁垒低：考点统计可被快速复制
- 依赖创始人IP：内容停更影响获客
- 登录有摩擦：必须关注公众号
- 无APP：移动刷题体验弱
- 解析深度未知：是否有详细解析不确定

## 二、CodeBrick 码砖（codebrick.tech）

### 2.1 产品概述

**CodeBrick 码砖**是一个面向计算机考研408统考的交互式可视化学习与刷题平台。Slogan："用可视化理解计算机科学"。

- 网址：www.codebrick.tech
- 创始背景：作者"考研时整理的408笔记，求职时写的刷题记录，工作后做成了交互式可视化"
- B站账号：codebrick（UID 282056930）
- 定位：408考研全科学习平台（非泛化编程刷题）

### 2.2 核心功能

1. **12张交互式数据屏（Dashboard）**：将2009-2026年18年真题（827题、189考点、26章节）可视化呈现
   - 大盘总览、矩阵、按题刷、按年练
   - 考频地图（圈出28个高频考点）
   - 章节图谱（按分值分布调整复习顺序）
   - 命题轨迹（发现"高频久未考"板块）
   - 难题榜（TOP 30）、错题本、雷达图
2. **210个算法可视化**：覆盖408全四科+复试机试
   - 数据结构 64个、操作系统 39个、计算机组成原理 45个、计算机网络 29个、复试机试 33个
3. **335篇长文笔记**：5个Blog站点，图文交互一体
   - 数据结构73篇、操作系统77篇、计组63篇、计网56篇、求职算法66篇
4. **408真题可视化题库**：每道真题配交互式动画演示
5. **OJ在线评测**：支持Python/Java，目前22道真题OJ
6. **408潜力测评**：3分钟15题快速自测，产出备考画像

### 2.3 商业模式

目前处于**免费增长阶段**：
- 核心内容免费、无需注册
- Dashboard 部分功能需登录（返回401）
- 无明确付费层/会员制
- 尚未建立商业化变现路径
- 主要通过B站内容引流

### 2.4 技术栈

- 服务器：nginx
- 前端：推测使用 D3.js / Canvas API 做可视化 + React 或 Vue
- 博客系统：静态站点生成器（URL 结构特征）
- OJ：后端有沙箱化代码执行引擎
- 反爬策略：非浏览器请求返回403

### 2.5 用户量与知名度

- B站有视频"6分钟把握一套408真题工具"
- 小红书/知乎几乎无存在感
- 用户量推测：**数千至低万级别**
- 处于早期阶段，依赖B站和自然搜索流量

### 2.6 优缺点

**优点**：
- 差异化极其鲜明：每道题配交互式动画，市场独一无二
- 内容量惊人：335篇笔记+210个可视化+827道真题+22道OJ（个人项目难得）
- 学习路径贴心：按"剩余备考时间"推荐不同策略
- 免费低门槛
- 将刷题从"做题"升维到"看数据做决策"

**缺点**：
- 反爬/鉴权激进：影响SEO和分享体验
- 知名度弱：小红书/知乎几乎没有存在感
- OJ题量少：仅22道
- 商业化不清晰：长期可持续性存疑
- 以选择题为主：大题练习支持可能不足
- 个人项目维护风险

## 四、对独立开发者的启示

### 4.1 三种不同策略

| 策略 | 代表 | 核心逻辑 |
|------|------|----------|
| **直接变现** | 真题墙 | 极简产品+会员付费墙，简单粗暴赚钱 |
| **漏斗变现** | 研可岸 | 免费工具引流→付费高客单价服务转化 |
| **内容驱动（未变现）** | CodeBrick | 极致产品体验积累用户，等待变现时机 |

### 4.2 可抄的点

1. **研可岸的"考频统计"维度**：将原始题目做一层数据分析加工，比单纯展示题目有更高价值
2. **CodeBrick的"可视化"理念**：交互式动画让抽象概念可触摸，极大提升学习体验
3. **真题墙的"抖音爆款引流"**：不是自己发视频，而是让KOL帮你推荐（苦瓜学长那条99.8万赞）
4. **研可岸的"公众号登录码"**：零成本实现私域沉淀，比传统注册登录更聪明
5. **CodeBrick的"按剩余时间推荐策略"**：场景化引导，提升用户感知价值

### 4.3 赛道选择建议

如果要做类似产品，建议：
- **数学方向**已有真题墙和研可岸瓜分 → 竞争激烈
- **408方向**CodeBrick做得深但未商业化，研可岸刚切入 → 有机会
- **其他科目**（英语/政治/法硕/CPA/法考）→ 蓝海，可直接复制模式

## 五、实时在线人数 → 总用户量的估算方法

### 5.1 推导公式

```
DAU ≈ 平均在线人数 × (有效时长 ÷ 平均会话时长)
MAU ≈ DAU ÷ DAU/MAU比（工具类通常 0.2-0.4）
注册用户 ≈ MAU ÷ 月活/注册比（通常 0.1-0.3）
```

### 5.2 实例（以真题墙镜像站"1人在线"为例）

| 步骤 | 计算 | 结果 |
|------|------|------|
| 有效时间 | 早8-晚12点 = 16h = 960min | 960min |
| 平均会话时长 | 考研刷题类约 15-30min | ~20min |
| 平均在线（非高峰） | 镜像站显示1-3人 | ~2人 |
| **DAU（非高峰期）** | 2 × (960/20) | **~96人** |
| 考研季倍数 | × 5-10倍 | **500-1000 DAU** |
| **MAU（考研季）** | 1000 / 0.3 | **~3300人** |
| **累计注册** | 3300 / 0.15 | **~2.2万人** |

### 5.3 注意事项

- 镜像站数据是下界，主站流量远高于镜像站
- 考研类产品有极强季节性（9-12月为高峰，是平时的5-10倍）
- "实时在线"数据容易受采样时间影响（凌晨 vs 晚高峰差异巨大）
- 工具类产品的"会话时长"比社交类短很多，换算时要注意
- 最可靠的估算是多数据源交叉验证（B站/抖音/搜索热度/实时在线综合判断）
