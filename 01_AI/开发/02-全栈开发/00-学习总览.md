# 全栈学习 5 天速成计划

## 技术栈全景图

```
你要学的东西，用盖房子来比喻：

HTML        → 房子的骨架（墙壁、门窗、房间划分）
CSS/Tailwind → 房子的装修（颜色、布局、好不好看）
JavaScript   → 房子的电路（开关灯、开关门、互动功能）
TypeScript   → 电路的安全检测（防止接错线）
Next.js      → 整栋楼的设计图纸（怎么盖、盖几层、电梯怎么走）
shadcn/ui    → 精装修套餐（现成的高质量家具，拿来就用）
Supabase     → 物业管理系统（用户管理、数据存储、安全门禁）
```

## 5 天学习安排

| 天数    | 主题                       | 核心目标                | 预计时长 |
| ----- | ------------------------ | ------------------- | ---- |
| Day 1 | HTML 基础                  | 能手写一个完整的个人简介页面      | 5-6h |
| Day 2 | JavaScript + TypeScript  | 理解变量、函数、DOM 操作，能做交互 | 6-8h |
| Day 3 | Tailwind CSS + shadcn/ui | 不写 CSS 也能做出好看的页面    | 5-6h |
| Day 4 | Next.js 核心               | 理解路由、组件、数据获取，能搭应用框架 | 7-8h |
| Day 5 | Supabase + 实战项目          | 连数据库，做一个完整的全栈小应用    | 6-8h |

## 学习方法建议

**第一，动手优先。** 每个知识点都配了可运行的代码，先复制运行看效果，再回头理解原理。不要试图先把概念全搞懂再动手——编程是手艺活，练出来的。

**第二，用好在线工具。** 不需要一开始就搭本地环境，这些在线工具可以让你秒看效果：
- HTML/CSS/JS 练习：https://codepen.io
- Tailwind 练习：https://play.tailwindcss.com
- Next.js 官方教程：https://nextjs.org/learn

**第三，遇到不懂的就问 AI。** 把报错信息、不理解的代码直接贴给 AI（CatDesk、ChatGPT、豆包都行），让它逐行解释。这比自己啃文档快 10 倍。

**第四，不要追求完美。** 5 天的目标不是精通，而是「能读懂 AI 生成的代码 + 能独立搭一个简单应用」。细节以后边用边查。
