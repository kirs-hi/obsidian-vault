## 题目2

```java
需求:
	现有一个整数数组{100,50,90,60,80,70}。请编写代码，计算数组中的所有元素的和并打印。
```



### 你的答案:

```java
int[] arr = {100,50,90,60,80,70}
int sum = 0;
for(int i = 0;i<arr.length;i++){
    sum+=arr[i];
}
```
