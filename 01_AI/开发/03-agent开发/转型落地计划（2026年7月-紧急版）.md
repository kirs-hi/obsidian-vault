## 第 2 周：面试八股 + 大量投递

### 面试八股（每天 2-3 小时，不要花更多时间）

你已经整理了尚硅谷面试题（在 `/Users/szx/Documents/red/01_AI/开发/03-agent开发/面试实战/` 目录下），直接用。重点只背这些：

**必考（每道都要会）：**

1. **手写 ReAct Agent Loop**（伪代码级别，不用框架）

```python
def agent_loop(query, tools, llm, max_steps=5):
    messages = [{"role": "user", "content": query}]
    for step in range(max_steps):
        response = llm.chat(messages, tools=tools)
        if response.tool_calls:
            for call in response.tool_calls:
                result = execute_tool(call.name, call.args)
                messages.append({"role": "tool", "content": result})
        else:
            return response.content
    return "达到最大步数"
```

2. **RAG 完整流程**
   - 文档加载 → 分块（chunk_size/overlap 怎么选）→ 向量化（embedding 模型选型）→ 存储（Chroma/Milvus）→ 检索（top_k + 重排序 Rerank）→ 生成（Prompt 拼接 + LLM）
   - 优化手段：HyDE、多路召回、Parent-Child Chunk、元数据过滤

3. **MCP 协议**
   - 是什么：Model Context Protocol，标准化 Agent 调用外部工具的协议
   - 三个核心概念：Tools（工具）、Resources（资源）、Prompts（提示模板）
   - 和 Function Calling 的区别：MCP 是跨模型/跨应用的标准协议，FC 是单模型 API 层面的

4. **LangGraph 核心概念**
   - StateGraph、Node、Edge、Conditional Edge
   - 和 LangChain AgentExecutor 的区别：LangGraph 是状态机，更灵活
   - 什么时候用 LangGraph vs 纯代码手写

5. **Prompt Engineering**
   - 结构化输出（JSON mode）
   - Few-shot、Chain-of-Thought
   - System Prompt 设计原则

**高频（尽量会）：**

6. Agent 设计模式：ReAct / Reflection / Plan-and-Execute / Multi-Agent
7. 向量数据库选型：Chroma vs Milvus vs pgvector，各自适用场景
8. Token 成本优化：上下文窗口管理、缓存策略、模型选择
9. Agent 的记忆系统：短期记忆（对话历史）vs 长期记忆（向量存储）
10. 幻觉处理：RAG 兜底、事实校验、置信度阈值

**可能问（知道就行）：**

11. Fine-tuning vs RAG 的选择
12. 多模态处理
13. Agent 安全（Prompt 注入防护）

### 面试中的话术

**"你为什么从数据开发转 AI Agent？"**

> 我在美团做数据开发的过程中，搭建了很多数据 pipeline 和自动化工具。后来我发现，这些 pipeline 的设计思路——数据流编排、异常处理、重试机制——和 Agent 的工作流编排非常相似。我开始用 LangGraph 重构一些内部工具，做了一个代码知识图谱的 Agent，效果很好。这让我确定了转向 AI Agent 开发的方向。

**"你是外包还是正式？"（如果被问到）**

> 我是通过外包公司入驻美团的，主要负责供应链数据开发。虽然身份是外包，但我参与的是核心业务系统，独立负责了实时数仓和数据质量治理两个项目。

不要撒谎，但也不要主动提。

**"你的 Agent 项目有上线吗？"**

> 目前是内部工具阶段，在团队内部使用。我用它来辅助代码审查和技术方案调研，效果是把代码理解的效率提升了大约 X 倍。

**"你对薪资的期望？"**

根据调研数据，你的合理区间是 1.5-2.5 万。面试时可以说：

> 我期望的范围是 18K-25K，具体可以根据岗位的具体情况和发展空间来谈。
