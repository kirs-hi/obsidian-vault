## 四、Agent 备考建议

### 必须能完整讲述的

1. **Agent 整体架构**：感知→规划→行动→记忆的循环，各模块的技术选型。
2. **LangChain / LangGraph**：核心概念（Chain、Agent、Tool、Graph、State）、适用场景、与原生实现的对比。
3. **Function Call / Tool Use**：OpenAI Function Calling 原理、如何定义工具Schema、错误处理、多工具编排。
4. **Multi-Agent 协作**：任务分配策略、通信机制、冲突解决、典型框架（CrewAI、AutoGen）。

### 必须有实战经验支撑的

1. **Prompt Engineering**：系统提示词设计、Few-shot构造、思维链引导、输出格式控制。
2. **ReAct 范式**：Reasoning + Acting 交替执行、观察反馈循环、异常恢复。
3. **MCP 协议**：Model Context Protocol 的作用、与 Function Call 的区别、接入流程。
4. **记忆机制**：短期记忆（对话上下文）、长期记忆（向量存储）、工作记忆的设计与管理。

### 加分项（体现深度）

1. Agent 可观测性：日志、Trace、评估指标体系搭建。
2. 复杂工作流编排：条件分支、并行执行、人机协作节点。
3. Agent 安全性：提示注入防护、工具调用权限控制、输出审查。
4. 生产环境部署：延迟优化、成本控制、降级策略。
