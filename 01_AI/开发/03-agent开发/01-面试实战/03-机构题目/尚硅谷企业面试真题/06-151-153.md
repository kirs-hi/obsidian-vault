## 152. 博彦科技

1. python中数值范围是什么，出现精度溢出是怎么解决的
 Python 的整数 `int` 采用任意精度实现，理论上没有范围限制，只受内存约束，所以不存在整数溢出问题。浮点数 `float` 遵循 IEEE 754 双精度标准，范围大约是 ±1.8×10³⁰⁸，有效精度约 15-17 位十进制数。
精度问题根因是十进制小数转二进制时的表示误差。对于比较操作，推荐用 `math.isclose` 设置容差；对于金融级别的精确计算，用 `decimal.Decimal` 模块，注意必须从字符串构造而非 float；对于科学计算大数累加，用 `math.fsum` 或 Kahan 求和算法补偿误差；对于有理数精确运算，用 `fractions.Fraction`。超出 float 范围的上溢/下溢，可以切换到 Decimal 或者在 log 空间运算。
2、科学计算使用的哪些库
> Python 科学计算有成熟的生态栈。底层是 **NumPy** 提供多维数组和向量化运算，**SciPy** 在其上封装了优化、积分、插值等科学算法。数据处理层用 **Pandas**，可视化用 **Matplotlib/Seaborn**（静态图）和 **Plotly**（交互图）。机器学习用 **scikit-learn** 做经典算法、**XGBoost/LightGBM** 做树模型。深度学习方向选 **PyTorch**（研究主流）、**TensorFlow**（工业部署）或 **JAX**（高性能函数式）。符号计算用 **SymPy**，图像处理用 **OpenCV/scikit-image**。这套生态在 CPU 层面依赖 NumPy 的 C 底层，GPU 层面通过 CUDA + PyTorch/TensorFlow 加速，满足从数据分析到深度学习全链路需求。
