## 四、微调备考建议

### 必须能完整讲述的

1. **LoRA 原理**：低秩分解思想、A/B矩阵设计、rank选择、与全参微调的对比、QLoRA的量化策略。
2. **SFT 流程**：数据准备→格式化→训练→评估→部署，数据质量 > 数据数量的原则。
3. **RLHF 全链路**：SFT → Reward Model → PPO/DPO 三阶段，每阶段的目标函数和训练细节。
4. **DPO vs PPO**：DPO 去掉 Reward Model 的简化思路、数学推导核心、各自的优缺点和适用场景。

### 必须有实战经验支撑的

1. **训练数据工程**：数据清洗、去重、配比策略、指令数据格式（Alpaca/ShareGPT）。
2. **过拟合处理**：early stopping、dropout、数据增强、正则化、学习率调度。
3. **分布式训练**：DeepSpeed ZeRO-1/2/3 的区别、FSDP、混合精度训练、梯度累积。
4. **量化部署**：GPTQ/AWQ/GGUF 原理与选型、量化对精度的影响、服务化方案。

### 加分项（体现深度）

1. 奖励模型训练：数据标注规范、Bradley-Terry模型、奖励 hacking 问题。
2. 对齐税（Alignment Tax）：对齐训练对模型能力的影响与缓解。
3. 持续微调与灾难性遗忘：EWC、经验回放等策略。
4. 微调 vs RAG vs Prompt 的技术选型决策框架。
