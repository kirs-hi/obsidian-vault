## 一、核心结论（先看这里）

基于 918 个真实在招岗位数据，针对你的背景（二本、美团外包、5年 Python 经验）的核心结论：

**你能投多少？** 聚焦方向 653 个岗位中，你有资格投递 357 个，占 **54.7%**。排除原因：硕士/博士要求刷掉 189 个（最大杀手），5-10年经验要求刷掉 71 个，实习岗刷掉 26 个，大厂刷掉 10 个。

**薪资现实是什么？** 可投岗位薪资中位数 **1.8 万/月**，均值 1.89 万/月。最集中在 1.5-2 万区间（占 27.2%），其次是 2-2.5 万（占 23.4%）。网上说的"AI Agent 月薪 40K"只是大厂数据，跟你无关。

**哪些岗位最匹配？** 高匹配（3-5年经验要求）153 个，可冲击（1-3年/不限）200 个。

**学历是不是最大障碍？** 不是。88.8% 的可投岗位只要求本科，9% 接受大专，2.2% 学历不限。真正卡你的是硕士要求的那 189 个（占总量 29%），但这些本就不是你的目标。

## 六、竞争格局

### 6.1 供需状况

2026 年春招 AI 人才整体供需比为 0.97（接近 1:1 平衡）。但核心 Agent 开发岗位的供需比约 0.39（2.5 个岗位对 1 个合格求职者），说明企业端是缺人的。

但"缺人"主要缺的是硕士/大厂背景/有完整项目经验的人。对于你的背景，竞争格局如下：

### 6.2 你的竞争对手

| 对手类型 | 估计占比 | 威胁程度 |
|---------|---------|---------|

### 6.3 投递到 offer 的漏斗

```
投递简历 100 份
    ↓ 简历通过率 ~15-20%（二本+外包会被部分 HR 过滤）
面试邀请 15-20 个

二面 6-8 个
```

**提高转化率的关键杠杆：**

1. 有可演示的 Agent 项目 → 简历通过率可提升到 25-30%
2. 能手写 Agent Loop → 一面通过率可提升到 60%
3. 简历突出 MCP + Neo4j 差异化技能 → 面试邀请率提升
4. 优先投 300 人以下公司 → 学历过滤更宽松

## 八、求职策略

### 8.1 目标公司优先级

**第一梯队（最推荐）：** 100-999 人的上市公司或 B 轮以上公司。学历要求"本科"但不卡一二本，薪资 1.5-2.5 万。代表：昆仑联通、千方科技、和利时、新港海岸、中科软。

**第二梯队：** 国企/事业单位的 AI 部门。薪资不一定高（1.2-2 万），但有 13/14 薪，稳定性好。代表：航天通安、中都数科、北排智慧水务、许继电气、京东方。

**第三梯队：** 20-99 人的天使轮/A 轮创业公司。薪资可能偏高（1.5-3.5 万），但风险也高。代表：森纵艾数、湃方科技、通合智能。

**第四梯队（过渡）：** 外包/驻场。薪资 1-2 万，但可以快速积累 Agent 项目经验。代表：软通动力、外企德科、法本信息。

### 8.2 简历策略

1. 写"美团 数据开发工程师"即可，不主动提外包身份。面试被问到再如实回答。
2. 突出 Python + MCP + Neo4j + Agent 学习项目，弱化传统 ETL。
3. 准备一个可演示的 Agent Demo 项目，这比任何文字描述都有说服力。
4. 技能栈列：Python / LangChain / LangGraph / RAG / MCP / Neo4j / Hive / Flink。

### 8.3 面试准备重点

1. 手写一个 ReAct Agent Loop（不用框架）
2. 设计一个 RAG pipeline 的完整架构
3. 解释 MCP 协议的工作原理，展示 MCP 工具开发经验
4. 准备"为什么从数据开发转 AI Agent"的故事线
5. 准备"如何用知识图谱增强 Agent"的技术方案

## 九、数据来源与方法论

### 9.1 数据采集

通过 CatPaw 浏览器自动化工具，在 2026年7月6日对智联招聘（zhaopin.com）进行系统性采集。搜索限定北京地区，使用 11 个聚焦关键词（AI Agent、AI应用开发、大模型开发、智能体开发、大模型应用、LLM开发、LangChain、RAG开发、AI开发、大模型工程师、AI工程师），每个关键词抓取全部有效搜索结果页（最多 5 页），共获得 70 个有效页面。

### 9.2 数据处理

使用 Python 脚本对原始页面文本进行结构化解析，提取岗位标题、薪资、经验要求、学历要求、公司名称、公司性质、公司规模、行业等字段。跨关键词去重（基于标题+公司名），最终得到 918 个独立岗位。

### 9.3 聚焦筛选

将 918 个岗位按标题关键词分为"核心方向"（Agent/AI应用/大模型等，279 个）、"相关方向"（AI工程/大模型工程师等，374 个）、"不相关"（265 个）三类。聚焦分析前两类共 653 个岗位。

### 9.4 用户匹配筛选

基于用户背景（二本本科、5年经验、非大厂目标），排除学历要求硕士/博士、经验要求 5-10 年、实习岗、大厂岗位，得到可投递岗位 357 个。

### 9.5 局限性

- 仅覆盖智联招聘一个平台（猎聘、BOSS直聘、前程无忧、拉勾均需登录，无法采集）
- 智联招聘在北京 AI 方向的覆盖率约 30-40%，实际市场岗位总量更大
- 部分岗位可能已关闭或更新，数据反映的是 2026年7月6日的快照
- 薪资为 JD 标注范围，实际面试谈判后可能有浮动
