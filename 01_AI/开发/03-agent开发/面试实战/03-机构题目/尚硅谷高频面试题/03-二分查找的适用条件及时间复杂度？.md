# 第3章 二分查找的适用条件及时间复杂度？

> 来源：尚硅谷大模型智能体之高频面试题 V2.1.5

---

3.  二分查找的适用条件及时间复杂度？

- **适用条件**：有序数组（升序或降序）。

- **原理**：每轮将搜索范围缩小一半，直至找到目标或范围为空。

- **时间复杂度**：O (logn)，空间复杂度 O (1)。

  1.  快速排序和归并排序的核心思想及复杂度对比？

  -------------------------------------------------------------------------------------------------------------
  算法         核心思想                                                       时间复杂度（平均）   空间复杂度
  ------------ -------------------------------------------------------------- -------------------- ------------
  快速排序     选择基准值，将数组分为小于和大于基准的两部分，递归排序子数组   O(nlogn)             O(logn)

  归并排序     分治思想，将数组拆分至单个元素，再合并两个有序子数组           O(nlogn)             O(n)
  -------------------------------------------------------------------------------------------------------------
