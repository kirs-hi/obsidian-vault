4.  **什么是 Python 中的生成器？

生成器是实现迭代器的简便方式，为含 yield 表达式的普通函数。其逐个生成值，可暂停和恢复执行，节省内存，适用于处理大序列数据场景。

这里需要注意的是推导式的三个基本的[]列表,{}字典/集合，()生成器表达式，列表和字典推导式不涉及生成器/迭代器，因为这俩都是直接生成完整内容并加载到内存是完整容器，生成器是特殊的迭代器，不会一次性加载所有元素。
