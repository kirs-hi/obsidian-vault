1.  八是CLI 工具：从初始化到部署

系统提供了一套完整的命令行工具：

- init：初始化项目，生成目录结构和配置文件

- train：训练模型，把 Flow 定义、对话样本打包成模型文件

- run：启动服务，提供 REST API 和 WebSocket 接口

- inspect：启动调试页面，可视化查看对话流程

整个流程非常顺畅：init 创建项目 → 编写 Flow 和 Action → train 打包模型 →
run 启动服务 → inspect 测试调试 → run正式启动。
