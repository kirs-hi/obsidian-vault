# 第2章 工作流为什么选择LangGraph？

> 来源：尚硅谷大模型智能体之高频面试题 V2.1.5

---

2.  工作流为什么选择LangGraph？

    1.  状态管理：LangGraph 提供 TypedDict
        状态管理，节点间状态传递更清晰

    2.  条件路由：支持复杂的条件分支，如商品名确认后的路由判断

    3.  并发执行：多路检索节点可以自动并发执行

    4.  可视化：可以打印工作流的 ASCII 图，便于调试

    5.  循环支持：支持循环结构，便于实现迭代优化
