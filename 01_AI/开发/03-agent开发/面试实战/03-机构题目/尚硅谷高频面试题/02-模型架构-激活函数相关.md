2.  模型架构-激活函数相关

    1.  补充：其他常用ReLU变体

        1.  Leaky ReLU

**公式**：$f(x) = \left\{ \begin{array}{r}
x,\ \ \ \ \ \ \ \ x \geq 0 \\
\alpha x,\ \ \ \ \ x < 0
\end{array} \right.\ $ (α为固定小常数,如0.01)

**特点**：负区间保留微小梯度，避免神经元失活。
