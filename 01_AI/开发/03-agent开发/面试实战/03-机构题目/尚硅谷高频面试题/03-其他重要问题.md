3.  其他重要问题

    1.  **解释 Python 中的 GIL（全局解释器锁）？

GIL 是 Python 解释器的一种机制，其保证任意时刻仅一个线程能执行 Python
字节码。它让 Python 多线程处理 I/O 密集型任务效率尚可，但执行 CPU
密集型任务时，多核心CPU只能发挥和单核CPU一样的性能。

绕过GIL的方式：使用多个进程，每个进程有自己的Python解释器和GIL，然后多进程并行。除此之外还可以使用Cpython或者C扩展，可以释放GIL锁，以及Numpy和SciPy等库也可以不受GIL限制。

**注意**：Python3.14的发布标志着GIL从强制变成了可选项。
