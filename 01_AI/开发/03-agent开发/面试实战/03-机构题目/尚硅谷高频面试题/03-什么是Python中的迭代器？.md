3.  什么是Python中的迭代器？

Python
里的迭代器就是能逐个取元素的对象，得实现__iter__和__next__两个方法，特点是懒加载
------
用的时候才生成元素特省内存，而且只能往前取、取完就不能复用了。列表、字符串这些可迭代对象，调
iter () 就能转成迭代器，for 循环遍历其实就是自动调 next ()
取元素，取完抛的异常会被自动捕获。迭代器主要就是用来处理大数据集，避免一次性加载占满内存的。
