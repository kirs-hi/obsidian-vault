2.  **数据安全层面**：

    - **敏感数据脱敏**：调用外部 API 前，通过 Coze 的 "数据处理节点"
      对商户名称、商品价格等敏感信息脱敏（如替换为占位符）；

    - **权限管控**：对接的第三方 API 仅授予 "只读 / 生成"
      权限，不开放商户原始数据的修改 / 删除权限；

    - **数据传输加密**：所有 API 调用均使用 HTTPS 协议，且在 Coze
      插件配置中开启 "请求参数加密"，避免数据传输过程中泄露。
