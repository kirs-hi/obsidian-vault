8.  独创科技（军工外包）

    一面

**"康威⽣命游戏"（Conway's Game of Life）**

题⽬描述：假设你有⼀个M⾏N列的⽹格（棋盘），每个格⼦被标记为"存活"（⽤1表⽰）或"死亡"（⽤0表⽰）。这个⽹格的排列被称为当前的"状态"，我们的任务是根据⼀组规则计算出下⼀状态。

**规则如下：**

1.
邻居：每个细胞有⼋个邻居，分别是：上、下、左、右，以及四个对⻆线⽅向。

2.
⼈⼝稀少（Underpopulation）：⼀个存活的细胞，如果周围存活的邻居少于等于1个，则在下
⼀状态中死亡。

3.
⽣存（Survival）：⼀个存活的细胞，如果周围有恰好2个或3个存活的邻居，则在下⼀状态中继续存活。

4.
⼈⼝过多（Overpopulation）：⼀个存活的细胞，如果周围有4个或更多存活的邻居，则在下⼀
状态中死亡。

5.
繁殖（Reproduction）：⼀个死亡的细胞，如果周围恰好有3个存活的邻居，则在下⼀状态中变
为存活。

**输⼊**

⼀个⼆维数组（board），表⽰当前的⽹格状态：

• 1表⽰该位置的细胞是"存活"的。

• 0表⽰该位置的细胞是"死亡"的。

**输出**

返回⼀个新的⼆维数组，表⽰⽹格的下⼀状态。

**⽰例：**

| board = [ [0, 1, 0], [0, 1, 0], [0, 1, 0] ]                   |
|                                                                       |
| new_board = next_board_state(board)                                   |
|                                                                       |
| for row in new_board:                                                 |
|                                                                       |
| print(row)                                                            |

输出：

| 1 [0,0,0]                                                           |
|                                                                       |
| 2 [1,1,1]                                                           |
|                                                                       |
| 3 [0,0,0]                                                           |
