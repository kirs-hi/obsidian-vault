# 海外独立创收日记(11)-996码农如何进行独立开发副业创收

> 来源：微信公众号「海外独立开发者日记」

## 为什么选独立开发

对于996码农来说，独立开发是最适合的副业方向，原因有几个：技术门槛低（本身就是程序员）、时间灵活（不需要固定时间）、投入小（几乎零成本启动）、收入可持续（做出产品后持续产生收益）、技能提升（产品/设计/运营全面成长）。

## 策略

### 利用现有技能快速上手

不要从零学新技术，而是基于现有技能延伸：

- Java 后端 → Android 原生开发
- 前端开发 → React Native / 浏览器插件
- Python → 桌面应用 / 自动化工具

### 选择合适的时间段

- 工作日晚上：2-3小时
- 周末：4-6小时

### 从小项目开始

- 工具类（计算器、转换器、记录器）
- 效率类（习惯打卡、待办清单、时间管理）
- 娱乐类（壁纸、音效、小游戏）

## 时间管理

- 工作日 19:00-22:00 集中开发
- 周末集中开发，做功能迭代和优化

## 技术选择

- 移动端：Flutter / React Native
- Web 端：Vue + Node
- 桌面端：Electron

## 收入变现

- AdMob 广告收入
- 应用内购买（一次性买断 / 订阅制）
- 企业版 / 高级功能

## 真实案例

- 我自己：Flutter 工具类 App，3个月开发，月入 $30
- 前端朋友：Vue 在线工具网站，月入 $200
- Java 朋友：Android 效率 App，月入 $150

## 建议

- 不要急着辞职，先跑通副业模式
- 从现有技能起步，不要从零学
- 小项目开始，快速验证
- 给自己 6-12 个月的耐心
